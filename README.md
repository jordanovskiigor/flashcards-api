# flashcard-api

Flashcard API built with Express and Mongoose.

## Quick start

1. Copy `.env.example` to `.env` and set `MONGO_URI`.
2. Install dependencies: `npm install`.
3. Start: `npm run dev` or `npm start`.

## Endpoints

- GET /api/sets
- POST /api/sets
- PUT /api/sets/:id
- DELETE /api/sets/:id
